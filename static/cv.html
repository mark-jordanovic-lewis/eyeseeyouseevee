About Me

Ruby is my language of choice in work. The ecosystem is rich, the syntax is flexible, there are very few web technologies which do not expose a Ruby API, and, it reads easily. I've been working in Rails for about five years solidly, I picked up Ruby as a scripting language to avoid learning perl when I was piping the output of molecular simulation code into analysis code I'd written in c++ and it's been with me ever since.

My interest as a programmer, up until recently, has been to feel out different styles and ways of expressing oneself in different languages to pin down a good way of approaching problems regardless of language paradigm. Recently I've settled into four main languages, Ruby, Rust, Elixir and Angular. I love learning new languages and I'd relish having an excuse to pick up another.

The non-language technologies I have chosen to focus on are Docker, Postgresql, rabbitmq (though my experience with rabbitmq is minimal at the moment) and LDAP. I definitely stack up on the creative side of the programmer spectrum, being an INTP type. I thrive when pair programming; collaborating and building new things closely with a partner, and, being left with a defined task to zone into.

I am proficient in git, have used a few CI/CD and monitoring tools and monitoring softwares such as raygun, logstash, grafana. I'm familiar with the AWS cli and web UI (but I don't like the web UI). My docker knowledge is not complete but I am learning more all the time.

Outside of work I have my personal learning, which has become using the set of technologies I have decided to stick to, rather than branching out into new things. I play guitar and sing, raise my daughter and work on my projects. I value freedom, family and supporting my peers above all other things.
